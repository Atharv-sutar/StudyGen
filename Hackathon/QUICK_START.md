# 🚀 StudyGen - Quick Start Guide

## Steps to Get Started (5 Minutes)

### Step 1: Open the Application
1. Navigate to the Hackathon folder
2. Double-click `index.html` to open in your browser
3. You'll be automatically redirected to the login page

### Step 2: Create Your Account
1. Click **"Register here"** on the login page
2. Enter your details:
   - **Username:** Choose any username (e.g., "student123")
   - **Email:** Any email address (e.g., "you@example.com")
   - **Password:** At least 6 characters (e.g., "mypassword")
   - **Confirm Password:** Type password again
3. Click **"Register"** button
4. You'll see "Registration successful! Please login."

### Step 3: Login
1. Enter your username and password
2. (Optional) Check "Remember me" to stay logged in
3. Click **"Login"** button
4. Welcome to your Dashboard! 🎉

## Main Features (Quick Overview)

### 📊 Dashboard
- View your tasks at a glance
- See today's progress percentage
- Check integrated Google Calendar

### 📝 Schedule Tasks
- Click "Schedule Task" in the sidebar
- Add new tasks with priority, subject, and due date
- Edit or complete tasks anytime
- Delete tasks you no longer need

### 📚 Study Materials
- Click "Study Material" in the sidebar
- Add links to Google Drive, YouTube, PDFs, etc.
- Track progress for each material
- Quick "Open" button to access materials

### 📈 Analytics
- Click "Analytics" to see your statistics
- View total tasks, completed tasks, completion rate
- See pie chart of task completion
- Check bar chart of time per subject

### 🚪 Logout
- Click "Logout" in the sidebar anytime
- Your data is saved for next login

## Common Tasks

### ✏️ Add a Task
1. Go to **Schedule Task**
2. Fill in:
   - Title: `Study for Math Test`
   - Subject: `Mathematics`
   - Description: `Chapters 4-6`
   - Due Date: `Tomorrow`
   - Priority: `High`
   - Duration: `2` hours
3. Click **"Add Task"**

### ✅ Mark Task Complete
1. Find the task in Schedule Task page
2. Click **"Complete"** button
3. Task will fade slightly showing it's done
4. Click **"Undo"** to mark as pending again

### 📁 Add Study Material
1. Go to **Study Material**
2. Title: `Physics Lecture Notes`
3. Link: Paste your Google Drive link or URL
4. Click **"Add Material"**
5. Click **"Open"** anytime to access it

### 📊 Check Your Progress
1. Go to **Analytics**
2. See at-a-glance statistics
3. View visual charts showing your progress
4. Check subject-wise breakdown

## ❓ Troubleshooting

### Q: I see "No tasks yet" message
**A:** You haven't created any tasks. Click "Schedule Task" to add your first task.

### Q: Invalid username or password error
**A:** Check your username and password. No spaces, case-sensitive password.

### Q: My data disappeared
**A:** Check if you cleared browser cache/cookies. Data is stored in browser.

### Q: Mobile menu doesn't close
**A:** Menu auto-closes when you click a link. You can also click the hamburger again.

### Q: Charts not showing properly
**A:** Create at least one task with a due date to generate chart data.

## 🔒 Data Storage

- All data stored in your browser (localStorage)
- Survives browser restart
- Clears if you clear browser data
- Each browser/device has separate data

### To Backup Your Data
1. Open Developer Tools (F12)
2. Go to Application → Local Storage
3. Screenshot or copy the `users` object

### To Export Tasks
Currently manual export only. In future versions, export feature will be added.
